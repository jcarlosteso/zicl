The language

Since one of our goals is to develop a DSL “emulating” ZIL, this entry presents the language we want to support. We will analyze real code from Zork I and explain what does everything mean, including some implementation details, if they add context.

Some initial decisions about how to shape our DSL will be made, but not yet about its implementation, though. That will be dealt with when we analyze the game engine.
